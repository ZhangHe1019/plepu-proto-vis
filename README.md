# EconViz Proto

A responsive, frontend-only React proto website structure for visualizing Economic Policy Uncertainty (EPU) in Poland. Users can explore data dashboards, view methodology information, and contact the EPU Poland team.  

Built with **React**, **Ant Design**, and **Redux** (for state management). The latest version for PLEPU visualisation can be seen: https://epu-index.pl/

---

## Features

- **Dynamic Dashboards:** Add or remove multiple data visualizers.  
- **Responsive Layout:** Works on desktop and mobile screens using Ant Design Grid.  
- **Methodology Section:** Overview of the data collection and analysis approach.  
- **Contact Section:** Information for partnership or inquiries.  
- **Frontend-only:** No backend required; can be deployed as a static site.

---

## Technologies Used

- **React** – UI components.  
- **Redux** – Centralized state management.  
- **Ant Design (antd)** – Component library for responsive UI.  
- **Ant Design Icons** – Icons for interactivity (menu, close buttons).

## Getting Started

### Prerequisites

- Node.js >= 18  
- npm >= 9 or yarn >= 1  

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/epu-poland-website.git
cd epu-poland-website
```


2. Install dependencies:

```bash
npm install
```

3. Start the development server:
```bash
npm start
```
